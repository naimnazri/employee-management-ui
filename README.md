# Employee Management UI

Employee Management UI is a frontend application built using Angular 20 and Bootstrap. It communicates with ASP.NET Core Web API to manage employee data.

## Technologies Used

* Angular 20
* TypeScript
* Bootstrap 5
* RxJS

## Features

* Display employee list
* Search employee
* Add employee
* Edit employee
* Delete employee
* Success notification
* Responsive table layout

## Running the Project

1. Clone the repository

```bash
git clone <repository-url>
```

2. Install dependencies

```bash
npm install
```

3. Start the application

```bash
ng serve
```

4. Open

```
http://localhost:4200
```

## Backend API

Default API URL:

```
https://localhost:7236/api/Employees
```
